About EventConnect

EventConnect is a web application that helps users discover events happening around the world. Search for concerts, tech meetups, workshops, sports events and more.

The app also provides public holiday information for every country in the world including all African countries.

APIs used:
Ticketmaster Discovery API — live events
Nager.Date API — public holidays
